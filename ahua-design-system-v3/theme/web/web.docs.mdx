A Theme Provider and a hook that provides the design tokens for React.

### Component usage

You can use useTheme hook inside the component:

```js live
() => {
  const theme = useTheme();
  return <div style={{ color: theme.primaryColor }}>Hello World</div>;
};
```

Or wrap your component in the ThemeProvider and consume it as CSS vars:

```js live
() => {
  const theme = useTheme();
  return (
    <ThemeProvider>
      <div style={{ color: 'var(--primary-color)' }}>Hello World</div>
    </ThemeProvider>
  );
};
```
